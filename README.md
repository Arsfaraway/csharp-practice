# C# Practice Tasks

Этот репозиторий используется для проработки как основ языка C#, так и отдельных тем и смежных технологий, таких как:
- **LINQ** (запросы, оптимизация, асинхронность)
- **SQL** (запросы, транзакции, оптимизация)
- **Алгоритмы и структуры данных**
- **Работа с коллекциями**
- **Асинхронное программирование**
- **Тестирование (xUnit, Moq)**
- **Работа с файлами и потоками**

Каждая тема оформлена в виде отдельных задач с примерами решений и тестами.
